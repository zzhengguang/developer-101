# Ubuntn Xenial 新手入门

#### 简介

[Ubuntu](http://www.ubuntu.com/)(乌班图) 是一个免费的操作系统。2004年10月20日开始提供给用户使用。


#### 版本选择

* Desktop(桌面版): 有图形界面，可以使用鼠标操作。
* Server(服务器版): 黑呼呼的只能用键盘打命令。

#### 更新周期

桌面版和服务器版本更新的时间是 6 个月

#### 支持时间
LTS（Long Term Suppot）长期支持版本

* 至少 18 个月的技术支持
* LTS至少 5 年即技术支持
